# Evaluate A News Article with Natural Language Processing

4th project at [Udacity](https://www.udacity.com/course/front-end-web-developer-nanodegree--nd0011) Front End Web Developer Nanodegree program.

This project intend  to build a web page to run Natural Language Processing (NLP) on articles o found on other websites. When a user submits a URL of an article



## Build Tools
* HTML
* CSS
* JavaScript
* Node
* Express
* Webpack
* meaningcloud API
* Jest
* Workbox

## Installation
Make sure Node and npm are installed from the terminal.
```
node -v
npm -v
```



Install loaders and plugins

# Choose the necessary installation for your development mode
npm i -D @babel/core @babel/preset-env babel-loader
npm i -D style-loader node-sass css-loader sass-loader
npm i -D clean-webpack-plugin
npm i -D html-webpack-plugin
npm i -D mini-css-extract-plugin
npm i -D optimize-css-assets-webpack-plugin terser-webpack-plugin
```

Start the project

Command | Action

`npm run build-prod` | Build project
`npm start` | Run project

 Open browser at http://localhost:8080/
